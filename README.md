# GradeUp — Rúbricas y Calificación

Herramienta de calificación de proyectos con rúbricas analíticas y holísticas, diseñada para docentes.

## ✨ Características

- **Rúbricas analíticas y holísticas** para evaluación detallada
- **Importación de rúbricas** en formato JSON y Excel
- **Importación de alumnos** desde NextUp (formato JSON)
- **Selector ruleta** de estudiantes con modo normal y rápido
- **Evaluación individual y por equipos** con ajuste de calificación por alumno
- **Comentarios** individuales y por equipo
- **Biblioteca de rúbricas** con opción de guardar plantillas
- **Exportación** a Excel y Word (.docx)

## 🚀 Uso

Abre directamente [`index.html`](./index.html) en tu navegador — no requiere instalación ni servidor.

También disponible en GitHub Pages: [Ver app →](https://TU_USUARIO.github.io/gradeup/)

## 📋 Formatos de importación

### Alumnos (desde NextUp)
```json
[
  {
    "grupo nombre": "3A",
    "alumno nombre": "María García",
    "alumno id": "12345"
  }
]
```

### Rúbricas (JSON)
Exportadas desde la propia aplicación.
